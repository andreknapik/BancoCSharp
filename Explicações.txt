Historia do Sistema.

1o Ser Possivel Criar Conta Corrente OK
2o Ser Possível Criar Conta Poupança OK
3o Ser Possível Criar Conta Investimento OK
4o Ser possível Efetuar depósito. OK
5o Ser possível Efetuar saque. OK
6o Ser possível Efetuar transferencia. OK
7o Ser possível Consultar Saldo. OK
8o Não deve ser possível criar uma conta bancaria sem tipo. OK
9o Cada tipo de conta possa implementar de forma diferente o metodo ImprimirExtrato OK



10o Ser possível Visualizar as Movimentações Bancárias.- OK

    10.2 Uma lista para armazenar essas movimentações. - OK
