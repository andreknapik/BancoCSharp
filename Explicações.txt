Historia do Sistema.

Fomos contratados para criar um sistema bancário onde fosse possivel criar as seguintes questões:

1o Ser Possivel Criar Conta Corrente
2o Ser Possível Criar Conta Poupança
3o Ser Possível Criar Conta Investimento
4o Ser possível Efetuar depósito.
5o Ser possível Efetuar saque.
6o Ser possível Efetuar transferencia.
7o Ser possível Consultar Saldo.
8o Ser possível Visualizar as Movimentações Bancárias.
